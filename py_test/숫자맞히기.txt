[[ 절차적 프로그램 연습 ]]
- 숫자 맞추기 게임
1) 콘솔 모드에서 진행
2) 프로그램이 구동되면 "게임의 제목 입력"
   이라고 출력됨
3) 입력하지 않고 엔터를 치면 오류를 출력하고 
   다시 입력을 요청한다.
4) 입력한 게임의 제목이 25자를 넘으면 오류를 출력하고 
   다시 입력을 요청한다
5) 입력을 받으면 아래처럼 정렬하여 출력
==================
=   게임 제목   ==
=   v1.0.0      ==
==================
6) 게이머의 이름을 입력받는다 
  (프럼프트:게이머의 이름을 입력하세요)
7) 입력않하면 오류
8) 게임시작
9) (프럼프트:0~99사이의 값으만 AI의 값을
    예측하여 입력하세요) 
10) 공백을 넣으면 오류
11) 숫자가 아니면 오류
12) 값의 범위를 넘으면 오류
13) 값이 작으면 작다고 출력하고 다시 입력받음
14) 값이 크면 크다고 출력하고 다시 입력받음
15) 시도 횟수를 기록
16) 최종 값을 맞추면 
    총 x번 시도만에 맞췄다고 출력하고
    다시 게임을 할것인지 물어본다
    다시하면 8번부터 시작
